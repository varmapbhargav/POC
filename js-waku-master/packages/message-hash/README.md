# @waku/message-hash

TypeScript implementation of the _Deterministic Message Hashing_ as specified in [14/WAKU2-MESSAGE](https://rfc.vac.dev/spec/14/).

See [JS-Waku README](https://github.com/waku-org/js-waku) for more information.
